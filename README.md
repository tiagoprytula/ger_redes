SNMPv2c
Send SNMP GET request using the following options:
with SNMPv2c, community ‘public’
over IPv4/UDP
to an Agent at demo.snmplabs.com:161

$ snmpget -v2c -c public demo.snmplabs.com 1.3.6.1.2.1.1.1.0 1.3.6.1.2.1.1.6.0
